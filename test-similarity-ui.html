How to Test

  1. Open the web UI at http://localhost:5173
  2. Look for the new toggle buttons next to the search bar (Text Search | Similarity)
  3. Click on "Similarity" to switch to semantic search mode
  4. Enter a query like "woodworking tools" or "typescript error"
  5. Observe the similarity scores displayed as colored percentage badges
  6. Click the "Similar" button on any document to find related documents
  7. Switch back to "Text Search" to use the regular search functionality
